Three assignments will be done in this course:

  1. Assignment 1 - Structured Information

    2. An XML schema has been created for a bookshelf. The schema contains details of each book, such as its name, author and publisher details. Along with this, it is indicated if the book is presently available or not. Different features of XSD, XSLT and XML have been used for this purpose.

  2. Assignment 2 - Social Media and Mashups

    3. We create a web application in which the front-end contains features from different social media services. The information received from the front-end is used in different ways by the server running in the back-end.

  3. Assignment 3 - Mobile software

    4. We will create a mobile interface to the application created previously and add special features.
